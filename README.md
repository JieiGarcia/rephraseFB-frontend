# rephraseFB (Frontend)

🌐 **[Live Demo (アプリを試す) ](https://rephrase-fb.netlify.app)**
※ゲストログイン（Try as Guest）から機能をお試しいただけます。

英語を学習する日本語母語話者（L2学習者）のための、リアルタイム・ライティング支援アプリケーションです。本リポジトリはフロントエンド（React）のソースコードを管理しています。

👉 **バックエンド（Go + PostgreSQL）のリポジトリは[こちら](あなたのバックエンドリポジトリのURLを貼る)**

## 🌟 主な機能

- **リアルタイム日本語検出**: ユーザーの入力から日本語や日英混在の文章を自動検出し、提案ボタン（💡）をリアルタイムに表示します。
- **AIによる文脈を考慮した提案**: ユーザーの入力意図を汲み取り、より自然な英語表現を提案します。
- **音声読み上げ (TTS)**: 提案された英文のネイティブな発音をワンクリックで再生できます。
- **2つのタスクモード**: 統制群（システムなし）と実験群（AI支援あり）の比較研究が可能です。

## 🛠 技術スタック (Frontend)

- **Framework**: React 19, Vite
- **Language**: TypeScript
- **Styling**: Inline CSS (Responsive Design)
- **Deployment**: Netlify

## 🚀 使い方（User Flow）

1. **ログイン**: 研究参加者はIDを入力。ポートフォリオ閲覧者は「Try as Guest」をクリック。
2. **タスク選択**: Control（支援なし）か Experimental（AI支援あり）を選択。
3. **ライティング**:
   - 日本語を混ぜて入力すると 💡 ボタンが出現します。
   - クリックで AI からの提案（文法・自然さ・明確さ）と解説が表示されます。
   - 「▶️」ボタンで英語の音声を再生できます。
4. **提出**: 最終的なテキストをSubmitし、バックエンドへデータを送信します。

## 💻 ローカルでの開発環境セットアップ

### 前提条件
- Node.js (v18+)
- npm

### インストールと起動

1. **リポジトリのクローン**
   ```bash
   git clone <repository-url>
   cd rephraseFB

2. **依存関係のインストール**
   npm install

3. **環境変数の設定**
   ルートディレクトリに .env.local を作成し、バックエンドのURLを指定します。
   VITE_API_URL=http://localhost:8080

4. **開発サーバーの起動**
   npm run dev

### 研究での活用について
本システムは、第二言語習得（SLA）および人間とAIのインタラクションに関する学術研究のために設計されており、ユーザーの提案受け入れ履歴やライティングのプロセスデータを収集・分析できる設計になっています。
